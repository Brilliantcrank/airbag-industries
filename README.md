# Airbag Industries

## Get Started

### Installing Hugo

Airbag is built on Forestry and Hugo. To install Hugo run:

```(shell)
brew install hugo
```

Verify the install:

```(shell)
hugo version
```

### Running the Local Server

Start the hugo server:

```(shell)
hugo server -D
```

or use NPM scripts:

```(shell)
npm start
```
